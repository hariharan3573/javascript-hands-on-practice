# JavaScript Practice

This repository contains JavaScript practical questions and hands-on exercises created to strengthen core JavaScript concepts through practice.

## What this repository covers

- JavaScript basics and fundamentals
- Array methods (map, filter, reduce, sort)
- String manipulation
- Functions and callbacks
- DOM manipulation
- Event handling
- Logical problem-solving exercises

## Purpose

The main goal of this repository is to practice JavaScript concepts by solving real and practical coding problems instead of only learning theory.

This repository is meant for:
- Learning by doing
- Improving problem-solving skills
- Revising JavaScript fundamentals
- Preparing for interviews and coding tests

## How to use this repository

1. Clone the repository
2. Open the files in a code editor (VS Code recommended)
3. Run JavaScript files in the browser or using Node.js
4. Modify the code and experiment to understand how it works

## Note

This repository is for practice and learning purposes.  
The code may be simple, experimental, or written in multiple ways as part of the learning process.
